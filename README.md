## Installation

```bash
$ yarn install
```

## Installation

```bash
$ yarn husky install
$ chmod ug+x .husky/*
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Seed role to MongoDB

```bash
yarn nestjs-command seed:role
```

## API Documentation

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
